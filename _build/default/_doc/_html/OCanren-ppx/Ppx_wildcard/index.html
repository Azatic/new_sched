Module Ppx_wildcard

This extension performs expansion of wildcards in:

  • Unification

    (q === __ % __)

    to 

    Fresh.two (fun __1 __2 -> q === __1 % __2)

  • Disequality

    (q =/= Std.List.cons __ __)

    to

    wc (fun __1 -> wc (fun __2 -> q =/= Std.List.cons __1 __2))
